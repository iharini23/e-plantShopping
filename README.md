Paradise Nursery is a React-based plant shopping application that allows users to browse a variety of plants, add them to a shopping cart, and manage their cart items easily.

## Features
- Browse plants by category
- View plant details such as name, image, description, and price
- Add plants to a shopping cart
- Increase or decrease plant quantity in the cart
- Remove plants from the cart
- View total cost of selected plants

## Technologies Used
- React
- Redux Toolkit
- JavaScript (ES6)
- CSS

## Project Purpose
This project is part of the IBM Developer Skills Network React course final assignment. It demonstrates component-based design, state management using Redux, and basic e-commerce functionality.
